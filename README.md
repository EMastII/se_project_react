# WTWR App

https://emastii.github.io/se_project_react/

I've developed a simple tool for users to access a page that uses location and weather data to suggest proper clothing attire at the time of request.

## 🚀 Features

- **Core Feature A:** I am using weather api's to grab weather data and suggest proper attire
- **Core Feature B:** I plan to add to the project so it has been structured and organized to add additional modal forms in the future via useState.
- **Core Feature C:** functions are able to be used across files efficiently to manage bandwidth and streamline the code for easy maintenance and updates.

## 📦 Tech Stack

- **Frontend:** React, HTML, CSS, JavaScript
- **Backend:** Node.js, package.json
- **DevOps:** GitHub Actions
